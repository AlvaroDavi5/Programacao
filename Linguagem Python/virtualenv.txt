
Para instalar o virtualizador de ambiantes, usar o comando:
	pip3 install virtualenv


Para criar um ambiente virtual, usar comando:	virtualenv [opções] <diretório>
	python3 -m venv diretorio

Para ativar um ambiente virtual, entrar no diretório criado e usar comando:
	source ./bin/activate

Para encerrar o ambiente virtual, usar comando:
	deactivate
