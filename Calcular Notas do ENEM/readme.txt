Os cálculos são feitos com base na Média Ponderada.

A Média é calculada multiplicando cada nota pelos seus respectivos pesos a elas atribuídos e dividindo a soma das multiplicações pela soma dos pesos.

Segue o exemplo:

Sendo N para "Nota", P para "Peso" e S para "Soma".

(P * N) + (P * N) + (P * N) + (P * N) + (P * N)
_______________________________________________
              P + P + P + P + P

ou

(P * N) + (P * N) + ... / Ps  =  Ns / Ps


Foram utilizadas as notas de corte, médias, notas mínimas e pesos dos cursos de Ciência da Computação e Ciências Biológicas para a realização do cálculo.
Todos os dados aqui utilizados foram extraídos do site " http://sisu.ufes.br/ ", no 1º e 2º processo seletivo do ano de 2019.
